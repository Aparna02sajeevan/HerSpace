### Project Description

HerSpace is a holistic wellness platform designed specifically for women's mental health and personal development. It combines mood tracking, journaling, financial planning, career resources, and mindfulness activities in one secure, supportive environment. The platform uses modern web technologies to provide a professional, welcoming space where women can prioritize their well-being.

### The Problem Statement

**Challenge:** Women face unique, interconnected wellness challenges in today's fast-paced world, yet lack integrated solutions to address them comprehensively.

**Key Issues:**
1. **Mental Health Crisis:** According to global health reports, women experience depression and anxiety at 1.5-2x the rate of men, yet struggle to access timely support
2. **Financial Insecurity:** Women earn 15-20% less than men globally and face barriers to financial literacy and independence
3. **Career Stagnation:** Gender gaps persist in tech, leadership, and STEM fields, with women lacking accessible mentoring and resources
4. **Emotional Exhaustion:** Societal pressures create stress around appearance, relationships, and life milestones
5. **Fragmented Solutions:** 
   - Mental health apps lack financial planning features
   - Journaling apps don't track mood patterns
   - Career platforms ignore wellness integration
   - Emergency resources are scattered across multiple platforms
6. **Privacy & Design Concerns:** Many wellness apps feel clinical rather than supportive; many lack a design philosophy centered on women's experiences

**Research Background:** Studies show that integrated wellness platforms increase user engagement by 40% and mental health outcomes by 30% compared to isolated tools.

### The Solution

**HerSpace: An Integrated Wellness Ecosystem**

HerSpace solves these challenges by providing a unified, beautifully-designed platform where women can manage all aspects of their well-being:

**Core Solution Components:**

1. **Mental & Emotional Wellness**
   - Daily mood tracking with pattern insights and trend visualization
   - Intelligent dashboard that suggests activities (meditation, music, affirmations) based on emotional state
   - Personal journaling with secure storage and offline access
   - Daily affirmations and confidence boosting (20+ unique messages with text-to-speech)
   - Emergency SOS button with one-click contact notifications

2. **Financial Empowerment**
   - Income and expense tracking with visual category breakdown
   - Real-time financial dashboard with Chart.js visualizations
   - Monthly spending analysis and budget tracking
   - Financial goal setting and progress monitoring
   - Data-driven insights to support financial independence

3. **Career & Personal Growth**
   - Curated learning resources (LinkedIn Learning, Coursera)
   - Job search filters and direct LinkedIn integration
   - Resume building templates and guides
   - Mock interview preparation resources
   - Professional network recommendations

4. **Holistic Wellness Activities**
   - Guided meditation sessions for stress relief
   - Curated wellness music playlists (Relaxation, Focus, Energetic, Sleep)
   - Stress-relief doodle canvas for creative expression
   - Rhythmic drumming for percussion therapy and anxiety relief
   - Goal tracking for personal development milestones

5. **Safety & Support**
   - Emergency contact management
   - Quick-call and SMS features for crisis situations
   - Crisis hotline information and mental health resources
   - Secure, private data storage with HIPAA-ready architecture

## Installation

### Prerequisites

- Python 3.14+
- pip (Python package manager)
- Git
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Setup Instructions

#### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/herspace.git
cd herspace
```

#### 2. Create Virtual Environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

#### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

#### 4. Configure Database
```bash
python manage.py migrate
```

#### 5. Create Superuser (Admin Account)
```bash
python manage.py createsuperuser
```

#### 6. Collect Static Files
```bash
python manage.py collectstatic --noinput
```

## Running the Application

### Development Server

```bash
python manage.py runserver
```

Access the application at: `http://127.0.0.1:8000`

### Production Deployment

For production deployment, use a proper WSGI server:

```bash
# Install gunicorn
pip install gunicorn

# Run with gunicorn
gunicorn herspace.wsgi:application --bind 0.0.0.0:8000
```

```bash

python manage.py runserver 8001
```

### Issue: Database migration errors
```bash
# Reset migrations (development only)
python manage.py migrate core zero
python manage.py migrate
```

### Issue: Static files not loading
```bash
# Collect static files
python manage.py collectstatic --clear --noinput
```

### Issue: "ModuleNotFoundError"
```bash
# Verify requirements are installed
pip install -r requirements.txt
```
